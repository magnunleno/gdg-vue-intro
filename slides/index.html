O Que é Vue?

  • Framework Progressivo para criar interfaces
  • Projetado para adoção incremental
  • Core focado na camada de visualização (view)
  • Facilmente integrável com outras bibliotecas JS
  • Perfeitamente capaz de produzir aplicações sofisticadas
    • SPAs (Single Page applications)
    • SSR (Server Side Rendering)
    • PWA (Progressive Web Apps)
